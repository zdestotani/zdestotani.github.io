About Me

Saya Zyka Destotani siswi SMK Negeri Kota Bengkulu Kelas 12 Jurusan Perancangan Perangkat Lunak Dan Gim (PPLG)

Saya telah belajar dengan tekun dalam perancangan perangkat lunak selama beberapa tahun terakhir, dan telah memperoleh pemahaman yang mendalam tentang konsep-konsep dasar dan teknologi terkini dalam pengembangan perangkat lunak. kemampuan saya meliputi UI/UX Design, data analyst, web developer, serta pemrograman dalam berbagai bahasa seperti python, java dan lainnya

Selain keahlian dalam perangkat lunak, saya juga memiliki minat yang kuat dalam dunia desain grafis. Saya memiliki pengalaman dalam menciptakan berbagai elemen desain, mulai dari logo, brosur, hingga desain web. Kreativitas dan ketelitian saya dalam desain grafis membantu saya menciptakan solusi visual yang menarik dan efektif.

Selanjutnya, saya juga tertarik dalam pengembangan web. Saya memiliki pengetahuan dalam HTML, CSS, dan JavaScript yang memungkinkan saya untuk membuat situs web yang responsif dan menarik. Kombinasi antara pemahaman perangkat lunak, desain grafis, dan pengembangan web memungkinkan saya untuk menciptakan proyek-proyek yang berfungsi dengan baik dan memiliki tampilan yang menarik.

Saya selalu mencari peluang untuk terus belajar dan berkembang dalam dunia teknologi informasi. Saya percaya bahwa pengembangan perangkat lunak dan desain web adalah bidang yang selalu berkembang, dan saya siap untuk menghadapi tantangan baru dan menggali potensi saya lebih dalam lagi.
